# QA Job Apply Agent

This AI agent automatically searches QA-related jobs, tailors your resume for each one (ATS > 90), and emails recruiters if contact info is available.

## Setup
1. Clone this repo
2. Add your OpenAI and email credentials as GitHub Secrets
3. Edit `config/settings.yaml` for your keywords
4. Upload your resume template to `/data`
5. The agent runs every 6 hours via GitHub Actions
